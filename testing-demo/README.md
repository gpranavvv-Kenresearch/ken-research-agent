## Frozen format (sample-vietnam-sugar.html)

- **Para 1** — data-loaded hook (numbers from word one), report link woven naturally, no Ken Research branding
- **Source line** — Ken Research homepage link + government/association sources only (no competitors)
- **Key Report Takeaways** — one line per segment cut
- **Trends and Insights** — Drivers + Restraints impact tables (modelled-estimate footnote)
- Hot-question H2s · 2 CTAs · "What X Must Do" · Conclusion · 5 FAQs · Ken Research branding paragraph
- Hard rules held: 8–9 links, all stats bold, zero em/en dashes

Status: format frozen 2026-06-03, pending changes.
